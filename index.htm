Introduction

Akshat Network Hub is a centralized platform which links all of my projects , skills , academic achievement , my scholastic records and many more in one place.

I am Akshat, a passionate student of Bachelor of Computer Application at MDU, Rohtak. This hub showcases my journey through academic excellence, personal growth, and professional development.

It link you to various sections including my projects, skills, achievements, and contact information.

Purpose

The purpose of Akshat Network Hub is to provide a comprehensive overview of my skills, achievements, and projects. It serves as a portfolio for potential employers, collaborators, and anyone interested in learning more about my work.

By centralizing all of this information, I aim to make it easier for others to connect with me and explore my contributions to the field of computer science.

Feel free to navigate through the various sections using the menu to learn more about me and my work. My goal is to continuously update and expand this hub as I progress in my academic and professional journey.

You can secuely navigate to all my projects and other sections using the sidebar menu. It uses redirect.js that store all trusted domains and Akshat network domain to avoid phishing attacks.

If you click external link , redirect.js trigger a confirmation dialog box to ensure you are navigating to a untrusted / trusted domain.

Enjoy exploring Akshat Network Hub!

Features :

  • Centralized platform for all my projects and achievements.
  • Responsive design for optimal viewing on all devices.
  • Secure navigation with redirect.js to prevent phishing attacks.
  • Regular updates to showcase new projects and skills.
  • Acts as a central hub for all my digital presence.
  • It showcase my academic records and scholastic achievements.

Tech Stack Used :

  • HTML5 for structuring the content.
  • CSS3 for styling and responsive design.
  • JavaScript for interactivity and dynamic content loading.
  • Redirect.js for secure navigation to external links.
  • Custom CSS Grid layout for flexible and adaptive design.

Project Cards :

Each project is represented as a card that includes a title, description, and link to the project details. The cards are designed to be visually appealing and easy to navigate.
